WinTask

Task manager.

Now by task manager, You might think its like Task Manager, System Monitor.
But nO! Its actually a command manager that allows you to store tasks in a single file so you do not have to have 600 .bat files.

The examples are on the github page

WinTask uses a singular JSON file.

Heck! Currently WinTask can allow you to make a JSON file for WinTask without even needing to write a A SINLE LINE OF JSON!
